# AGBootstrap

## Overview
This package implements the **Adaptive Generative Bootstrap (AGB)** method for small sample inference. It is designed to correct coverage probabilities in multimodal or small datasets.

## Installation
You can install the development version from GitHub using:

```r
# install.packages("devtools")
devtools::install_github("saifldeenalrefaee-blip/AGBootstrap")
```

## Example
Here is a basic example of how to use the package:

```r
library(AGBootstrap)

data <- c(576, 635, 558, 578, 666, 580, 555, 661, 651, 605, 653, 575, 545, 572, 594)
result <- agb(data)
print(result$ci)
```

## Reproducibility (Paper Simulation)

The simulation study reported in the paper can be reproduced from this repository.

**Configuration**
- Monte Carlo replications: `M = 500`
- Bootstrap iterations: `B = 10000`
- Seed: `123`
- Confidence interval: 95% percentile CI (`alpha = 0.05`)
- Scenarios:
  - **Normal** (main-text benchmark)
  - **MixtureSymmetric**, **HeavyTail_t3**, **Contaminated**, **SkewedLognormalCentered** (Appendix C)

**How to reproduce the outputs**
1. Clone the repository and open the project in RStudio (repository root contains `DESCRIPTION`).
2. Run the script from the repository root:

```r
source("analysis/scripts/run_simulation_B10000.R")
```

3. The script writes:
- Tables to `analysis/outputs/`:
  - `Simulation_Results_Table_B10000_Extended.csv`
  - `Simulation_Results_Wide_B10000_Extended.csv`
- Figures to `analysis/figures/`:
  - `Figure_3_Coverage_Convergence.png`
  - `Figure_4_Interval_Length.png`
  - `Figure_S_*.png` (supplementary coverage plots for Appendix C scenarios)

**Note**
The repository tracks the simulation script for reproducibility. Generated outputs (CSV/PNG) are not committed and can be regenerated by rerunning the script.
